Bay Area Friends of Tibet — Legacy Website Archive

This repository preserves the former website of Bay Area Friends of Tibet (BAFoT), a complete static mirror of the original Drupal site captured in June 2026.

The new BAFoT website is now live at www.friends-of-tibet.org.

This archive is served at archive.friends-of-tibet.org via GitHub Pages, so that decades of BAFoT's history — March 10th commemorations, campaigns, event records, and photo galleries dating back to the organization's founding — remain publicly accessible.

About this project

This archive was built and is maintained by Starshiner, a 501(c)(3) nonprofit, as part of its technology program providing infrastructure support for Tibet advocacy and Chinese human rights projects.

Technical notes


Static mirror generated with HTTrack; all internal links rewritten as relative paths
Deployed via GitHub Pages with a custom subdomain
Content is preserved as-is for historical reference and is no longer updated



湾区图博之友 — 旧版网站存档

本仓库保存的是湾区图博之友(Bay Area Friends of Tibet, BAFoT)的旧版网站,为原 Drupal 站点于 2026 年 6 月抓取的完整静态镜像。

BAFoT 新版官网已正式上线:www.friends-of-tibet.org。

本存档通过 GitHub Pages 托管于 archive.friends-of-tibet.org,以保留 BAFoT 数十年的历史资料——历年三月十日纪念活动、行动倡议、活动记录与图片集——供公众持续查阅。

关于本项目

本存档由 501(c)(3) 非营利组织 Starshiner 开发维护,是其科技项目的一部分,致力于为支持图博(西藏)与中国人权的项目构建基础设施。

技术说明


使用 HTTrack 生成静态镜像,站内链接均已改写为相对路径
通过 GitHub Pages 部署,绑定自定义子域名
内容按原样保存,仅作历史参考,不再更新
